# Declarative Deployment of Analytical Applications in GCP

This repository holds code related to the master thesis with the above title.  
Created by Martin Kadlec, year of defense is 2026.  
It builds on a code developed by a student team Null Hypothesis, of which I was 
a team leader during the initial implementation phase. The original work if the team
can be accessed in repo here: https://gitlab.com/R4tmax/dp_automateddwh . In this repository,
a completly new addition of Terrafrom based CI/CD system is provided, this repository has been created as
--mirror clone of the original one, and as such retains its git history.

Purpose of my thesis, and as such this repository, is to design, create and execute robust CI/CD system for cloud computing applications
utilizing common analytical use cases, such as DB Integrations and AI models, the idea is to address
lack of dedicated and comprehensive works, which provide in depth explanations, rationale
and baseline for others to utilize and build upon in their system.

## Environment consistency disclaimer
Since every GCP artifact is monetized, all environments will be decommissioned pending the 
acceptance of the thesis by the university, to prevent incurring running costs on dummy implementation.
